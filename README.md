# Madhav-Ecommerce-Sales-Dashboard
Power bi Dashboard
Project Objective:-
To analyze ecommerce sales data and build an interactive Power BI dashboard for tracking sales, profit, customer trends, and payment insights.

📈 Key Insights:-
Maharashtra had the highest sales.
Clothing category recorded maximum quantity sold.
COD was the most preferred payment mode.
Profit varied across different months.
Printers and Bookcases were among the most profitable sub-categories.

Power BI
